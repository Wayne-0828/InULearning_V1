📋 手動測試步驟

首頁 → 註冊流程

  1. 開啟首頁 (index.html)
  2. 點擊導航欄的 "註冊" 按鈕
  3. 確認跳轉到註冊頁面
  4. 或點擊英雄區的 "立即註冊"
  5. 測試註冊功能完整性

登入 → 註冊流程

  1. 開啟登入頁 (login.html)
  2. 滾動到頁面底部
  3. 點擊 "前往註冊" 連結
  4. 確認跳轉到註冊頁面
  5. 測試註冊後回到登入
