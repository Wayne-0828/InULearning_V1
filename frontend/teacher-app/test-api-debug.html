📊 測試結果

🔧 手動測試

您也可以在瀏覽器控制台中手動測試：

// 檢查 API 客戶端
console.log('API 客戶端:', window.apiClient);
console.log('API 客戶端 baseUrl:', window.apiClient?.baseUrl);

// 測試 API 客戶端
if (window.apiClient) {
    window.apiClient.get('/relationships/teacher/removed-students')
        .then(data => console.log('API 客戶端成功:', data))
        .catch(err => console.error('API 客戶端失敗:', err));
}

// 測試直接 fetch
fetch('/api/v1/relationships/teacher/removed-students')
    .then(res => res.json())
    .then(data => console.log('直接 fetch 成功:', data))
    .catch(err => console.error('直接 fetch 失敗:', err));
