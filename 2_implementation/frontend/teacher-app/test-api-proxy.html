🔧 手動測試

您也可以在瀏覽器控制台中手動測試：

// 測試 API 客戶端
console.log('API 客戶端:', window.apiClient);

// 測試直接請求
fetch('/api/v1/relationships/teacher-class')
    .then(res => res.json())
    .then(data => console.log('直接請求結果:', data))
    .catch(err => console.error('直接請求錯誤:', err));
