測試 2: 平方根

求解方程式：$x^2 = 16$

答案：$x = \pm 4$

測試 3: 複雜數學表達式

計算：$\sqrt{a^2 + b^2}$ 當 $a = 3, b = 4$ 時

解：$\sqrt{3^2 + 4^2} = \sqrt{9 + 16} = \sqrt{25} = 5$
