🚀 主要功能特色

🎯 智能 API 整合

自動檢測 API 可用性，支援真實資料和模擬資料無縫切換，確保系統在任何情況下都能正常運作。

📊 雙視圖模式

提供表格視圖和卡片視圖兩種顯示模式，滿足不同使用習慣和螢幕尺寸的需求。

🔍 即時搜尋篩選

支援學生姓名搜尋、班級篩選、分數排序等多維度篩選功能，快速找到目標學生。

📱 響應式設計

完美適配桌面、平板、手機等各種設備，提供一致的用戶體驗。

📈 統計概覽

即時顯示學生總數、平均分數、活躍學生數等關鍵指標，幫助教師快速了解班級狀況。

🎨 現代化 UI

採用現代設計語言，清晰的視覺層次，優雅的動畫效果，提升用戶體驗。

📋 使用說明

1. 點擊上方按鈕開啟學生管理頁面
2. 系統會自動檢測 API 連接狀態
